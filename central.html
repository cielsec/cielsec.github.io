Central de Ferramentas

Ferramentas de rede e OSINT leve — rápido, limpo e direto.

GeoIP

Consulta geolocalização do IP (cidade/país/ASN/org). Ideal pra triagem rápida.

  • Entrada: IPv4
  • Saída: localização + ASN/ORG + timezone
  • Dica: salva resultados no teu log / print pro relatório

IP Tracker

“Tracker” no estilo OSINT leve: GeoIP + RDAP + reverse DNS (quando disponível).

  • GeoIP (contexto)
  • RDAP (registro / handle / status)
  • Reverse DNS (PTR)

DNS

Resolve registros comuns (A/AAAA/MX/NS/TXT/CNAME/SOA/PTR).

  • A/AAAA: endpoints
  • NS/SOA: infra
  • TXT: SPF/DMARC/validações
  • PTR: reverse

WHOIS / RDAP

WHOIS moderno via RDAP (mais compatível com browser e menos dor com CORS).

  • Domínio: registrar/registrant (quando exposto), eventos, nameservers
  • IP: handle, status, entidades e datas
  • Bom pra relatório e validação de ownership
